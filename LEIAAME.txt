🎯 COMEÇAR AGORA (3 cliques)
═══════════════════════════════════════════════════════════

1. Clique duas vezes em:
   → START.bat

2. Escolha uma opção:
   
   Opção 1: COM SERVIDOR (recomendado)
   - Instala Node.js automaticamente se necessário
   - Executa server em http://localhost:3000
   - Dados persistem entre sessões
   
   Opção 2: SEM SERVIDOR (rápido)
   - Abre o dashboard imediatamente
   - Não precisa de instalações
   - Dados no navegador (perdem se fechar cache)

3. Pronto! O sistema está rodando 🚀

❓ PERGUNTAS FREQUENTES
═══════════════════════════════════════════════════════════

P: Preciso instalar algo?
R: Não se usar a Opção 2 (Quick Test). Para Opção 1,
   Node.js será instalado automaticamente pelo script.

P: Meus dados são salvos?
R: Na Opção 2, os dados ficam no navegador. Se limpar
   cache, perde tudo. Com Opção 1 (servidor), dados
   ficam no servidor.

P: Posso usar em múltiplos computadores?
R: Sim! Apenas com Opção 1 (servidor). Basta acessar
   http://[ip-do-servidor]:3000

P: Como crio uma nova empresa?
R: 1. Acesse http://localhost:3000/admin.html
   2. Preencha dados da empresa
   3. Clique em "Criar"
   4. Depois entre em http://localhost:3000/login.html

P: Esqueci a senha, como resetar?
R: Para Opção 2: Limpe o localStorage (DevTools > Storage)
   Para Opção 1: Crie novo usuário em /admin.html
